## Description

This playground is based on official docs in NestJS website

## Installation

```bash
$ npm install / yarn 
```

## Development

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Useful commands
```bash
# First Project Setup
$ npm i -g @nestjs/cli
$ nest new [project-name]

# Generate Controller
nest g controller [controller-name]

# Generate Providers - Service
nest g service [service-name]
```
